# Design System Overview

- **Tokens**
  Color, spacing, radii, type scale, shadows, motion. AA contrast; dark/high-contrast.

- **Type**
  H1/H2/H3 hierarchy, single primary CTA per screen.

- **Components (primitives)**
  Button, Input, Select, Badge, Card, Sheet/Dialog (from shadcn/ui later), Nav, Footer.

- **Motion**
  Tokenized durations/easings; respect `prefers-reduced-motion`.

- **Accessibility**
  Semantic landmarks, one H1, labels, keyboard-first, visible focus.
